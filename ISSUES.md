## How to Report Issues

When adding an issue, include:
1. **Platform**: macOS or iOS/iPadOS
2. **Description**: Clear, concise summary
3. **Steps to Reproduce**: Numbered steps
4. **Expected vs Actual**: What should happen vs what does happen
5. **Status**: Open, In Progress, Fixed, Won't Fix

## Issue Severity Guide

- **Critical**: Game won't launch, crashes, data loss, blocks gameplay
- **Major**: Significant gameplay issues, missing features, bad UX
- **Minor**: Cosmetic issues, edge cases, nice-to-have fixes
